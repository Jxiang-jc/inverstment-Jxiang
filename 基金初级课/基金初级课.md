# 一蓑烟雨任平生

### 从名字了解基金

- 基金的名字大多由3部分组成： 公司名称 + 投资方向 or 基金特点 + 基金类型

  - 通俗点说，就是这只基金从哪里来，投资了什么，还有他的的类型是什么。

  - ```js
    // 天弘安康养老混合 代码 420009
    	天弘基金管理有限公司 养老方向
    // 国泰沪深300 代码 020011
        国泰基金管理有限公司 跟踪"沪深 300"指数
    // 嘉实沪港深精选股票 代码 001878
        嘉实基金管理有限公司 沪港深
    ```

- 基金名字后面的ABC

  - 货币基金
    - 如果基金的名字里面带A的后缀，一般申购门栏比较低，面向的是普通的投资者。
    - 后缀为B的，一般申购门槛都比较高，主要面向的是资金量大的用户或者机构投资者。
    - 后缀为C的，可以理解为和B的一样。申购门槛比较高，B和C区别不明显。
    - 如果出现了D类，要么即使申购门槛更高了，要么是某个公司指定了一个交易平台，专门办理申购赎回，然后基金名字后面加了个D。
    - 如果出现E。表示这个基金只在指定的机构进行代销，比如互联网平台；还有一种情况是它是交易型货币基金，是场内交易的。
  - 债券基金的`abc`主要在于收费的方式不同。
    - 他们都是根据申购费，收取的时间不同来划分。
    - 债券基金中A类，一般是`前端收取申购费`。就是买的时候就收费。
      - 基金公司有时为了吸引土豪加入，会对申购金额大的提供一定幅度的优惠
    - B类呢。一般是`后端收取申购费`，就是买的时候先不扣费。等赎回的时候才一次性收取。费率则随你持有基金的时间长短来决定
    - C类呢。一般会免去申购费。但是很有可能这只基存在"销售服务费率"。也就是用一个费率代替另一个费率而已。
    - 总结：如果是短期投资（1-2年），应该选择C类债券。如果是长期持有（5年或5年以上）可以选择B类。如果是不知道投资多久。那就选择A类把。实际中，还是要根据实际情况比较再做选择噢

## 三山半落青山外

#### 筛选货币基金的四大指标

- `收益率`
  - 一般情况下，货币基金的收益率可以达到2%-4%。
  - 货币基金可以通过"每万份收益"和"7日年化"两个指标判断
    - 万份收益：就是指按上一日或上一个交易日的收益率，一万元本金能够赚多少钱。
      - 投资1千元。当天盈利0.2元。（`万份收益 = 0.2 / 1000 * 10000 = 2`）
    - 七日年化收益则是指，根据过去7天的收益总和，计算出的年花收益率。（）
      - 投资1000，过去七天赚了1元。（`七日年化收益 = 1 / 7 * 365 / 1000 * 100% ≈ 5.21%`）
    - 一般来说，货币基金的万份收益再1左右，7日年化在3%左右。
- `流动性`
  - 货币基金的流动性是指如果我们卖出这支基金后，自己多久能回到我们的账上。
  - 一般来说，有T+0 和 T+1两种交割方式。
- `成立时间`
  - 货币基金的成立时间，3-5年以上比较好
  - 如果一只基金太年的话，就没有历史的收益率等数据作为参考，从而对它做出合理的评估。相比较一些老牌基金，年轻的基金没有经过市场的磨练，隐藏的风险蛮大的。
- `基金规模`
  - 基金规模适中最好，一般不小于20亿，最大不要超过2000亿。货币基金的规模越大，就意味着这只基金中的钱越多
  - 规模越大的货币基金，有充足的资金应对赎回，也能留有更多的资金来投资
  - 规模越大的货币基金，因为资金问题无法正常运作而被清盘的风险也会比较低
  - `但是`规模太大的货币基金，可选择的投资品受限，收益反而会下滑。

  - 成立时间在3年以上的，更加稳健
- `基金规模`
  - 混合基金规模适中即可。建议2亿-100亿之间。
- `手续费`
  - 越低越好
- `基金经理的更换频率`
  - 越低越好
- `基金公司的盈利能力`
  - 一是看它在同类基金公司中的排名。只需要选择排名靠前的即可
  - 二是看它盈利是否跑赢沪深300指数，因为沪深300可以大致代表大盘的走势
- `基金经理的选股择时能力`（排除指标）
  - 基金经理的选股择时能力，主要参考四个指标：从业情况、评分、星级和管理基金的收益排名
  - 从业情况主要看这个基金任职多久，如果小于3年不建议挑选。评分高于50分比较有保障，星级越高越好，四级以上最好。管理的基金收益排名没有跑赢同类平均水平的。也可以排除。

- `纯债基金的买卖时机和“市场利率”密切相关。`

  - 市场利率： 是市场资金借贷成本的真实反映，能够及时反映短期市场利率的指标有银行间同业拆借利率、国债回购利率等。

  - 简单说：就是纯债基金的买卖时机和市场利率有关就好。

  - 其实，市场利率完全可以参考市场无风险收益率。也就是`在当前市场上，你把钱借给别人，在没有风险的情况下可以获得收益率。`

  - 市场利率一般参考10年期国债利率。（一下市场利率代表10年期国债利率）

    - 这是因为中长期债券可以更加细致的划分为中期债券和长期债券，10 年是中期债券和长期债券的分界点，期限短于10 年的债券收益较低，而超过 10 年的债券买的人又很 少，10 年期刚刚好，市面上几乎所有的投资品类都会跟它比较，所以 10 年期国债利率是一个非常有代表性的利率。

    - `市场利率上升，债券价格下跌;`

    - `市场利率下降，债券价格上升；`

    - ```js
      // 举个例子
      比如今年 2 月市场利率是 3%，二狗公司发行了债券，利息4%；到了 6 月份，市场利率升高到 3.5%，没有风险就可以获得 3.5%的收益，二狗公司的债券 4%的收益看起来没啥吸引力了呢。有些人就会觉得安稳获得 3.5%更划算，那
      么，那些买了二狗公司债券的人就会卖出债券，直到它的收益率重新变得有吸引力为止。从这个小例子就可以看到，随着市场利率从 3%升到3.5%，债券的价格是下跌的。同样的道理，市场利率下跌时，债券的价格会上涨
      ```

    - 那么当我们投资纯债基金的时候，肯定希望买入之后债券价格升高，也就是希望接下来市场利率下降。

    - ```js
      根据历史统计，中国 10 年期国债收益率的平均值在 3.3%-3.5%，这就是市场利率的平均值，由于市场利率波动小，比较稳定，这个平均值也是长期稳定的。
      所以我们只要拿现在的市场利率（也就是 10 年期国债收益率）和 3.5%比较就可以判断是否应该买入纯债基金：
      ```

    - `当前市场利率（等于 10 年期国债收益率）>3.5%，买入纯
      债基金；`

    - `当前市场利率（等于 10 年期国债收益率）<3%，卖出纯债
      基金，买入货币基金`

## 五陵少年争缠头

#### 指数基金

- 被动型基金
  - 基金经理不主动寻求超越市场表现，选取特定的股票作为投资对象，试图复制整个市场的收益率。长期来看，相比较主动型基金，被动型基金的收益更高更稳定
- 指数
  - 指数，其实就是一个选股规则，它按照某个规则挑选出一篮子股票，由专业机构通过复杂的计算，得出一个代表这篮子股票的平均价格，然后用来反映市场上，这一类股票的价格水平
  - 我们把这个平均价格称为"指数点位"，把放在篮子里的这些股票，称为这只指数的成分股。
  - 指数的成分股事可以变化的，一单某只股票不能满足原先的规则，就会被替换掉
- A股市场主要指数
  - 上证50指数
  - 沪深300指数
  - 中证500指数
  - 创业板指数
  - 红利指数

#### 指数基金的优点

- 灭绝人性
  - 只需要直接跟踪和复制指数
- 永垂不朽
  - 组成指数的公司不符合标准了，就会有新的公司代替
- 笑到最后
  - 只要经济是持续向好的，指数基金就有投资的价值

#### 指数基金的误差

- 指数基金本质是股票型基金，它依然存在风险，虽然是跟踪和复制指数来调整篮子里的股票组合。但是基金经理如果复制得有偏差，或者没有及时跟着指数调整，也会带来收益方面的误差风险，指数基金规模小或者运作历史不长，也会有风险
- 在天天基金网上，指数基金详情中可以看到`跟踪误差`，越小越好



## 六朝如梦鸟空啼

#### 指数基金的分类

- 宽基指数
  - 覆盖的行业更多更广，分配更加均匀，所以盈利也更加稳健。
    - 完全复制指数基金
      - 追求获得和跟踪指数持平的收益。
    - 增强型指数基金
      - 基金经理加入了自己的一些主观投资，追求超过跟踪的指数的收益。
- 行业指数（新手不建议）
  - 需要考虑整个行业的发展特点与阶段，受行业影响大，所以风险也会更高。



#### 选择指数基金

 - 确定想要的指数基金
   	- 指数本身没有好坏之分，只有合适与否
 - 精选靠谱的基金公司
   	- 选择TOP 10的基金公司.无论是资金还是稳定性、服务怎么样。大规模的基金公司都是首选。优先选择规模1000亿以上的公司
 - 锁定指数基金
    - 跟踪误差率
      	- 越低越好
    - 基金规模
      	- 基金规模不宜小于2亿
    - 成立年限
      	- 最好3年以上
    - 费用成本
      	- 管理费用率越低越好

## 七香车碾瑶墀月

#### 通过市盈率和市净率计算估值

- 市盈率（PE）= 公司市值/净利润，指的是一笔钱投资进一只股票，需要多长时间能绘本。简单理解就是，与一直股票的盈利能力相比，目前它的价格是不是够便宜。
  - 可以通过市盈率初步断定一个公司是贵还是便宜，市盈率越低，公司越便宜，市盈率越高，公司越贵。**但是不能作为唯一指标**
  - PE作为估值指标，简单粗暴。但是也有明显的缺陷。首先不同国家地区，不同的指数，估值的波动区间不一样。
  - 其次，就算在同一国家或地区来看，不同的行业市盈率各有不同，也没有可比性。
- 市净率（PB）= 市值/净资产，体现的是公司的净资产是否被低估。
  - 但是像微软这种以知识产权为产品的轻资产公司，不需要投入太多的资产，就能获得更多的利润，在这种情况下，仅凭市净率高，就断定公司是高估太武断
- 总结：
  - `以上方法估值，都不太靠谱`

- #### 应该结合 PB 和 PE 来计算才是最稳妥

  - **计算方式**

> 温度 = ((当前PB在历史数据中的百分比排位 \* 100) + (当前PE在历史数据中的百分比排位 \* 100)) / 2
>
> 历史数据一般去过去10年即可

- 温度越高，说明股市越火爆，上升空间越少，风险越高
- 温度越低，说明股市越冷淡，上升空间越大，风险越低

> 0 - 10° 股票市场萎靡不振（最佳买入时机） / 10 - 20° 市场逐步回暖（买入良机）
>
> 20 - 30° 股票市场继续走好（可以继续买入） / 30 - 40° 市场气氛热烈 （持有观望）
>
> 40 - 50° 市场过度热烈（应考虑逐步卖出）

> **温度高低和价格高低没有必然关系，温度反映的是当前的价格是被高估还是被低估**

- 历史数据可以在[理杏仁](https://www.lixinger.com)中导出

#### 一个Python自动算温度的脚本

```py
#!/usr/bin/env python3

import pandas as pd
import numpy as np

pd.set_option('precision', 2)
pd.set_option('display.unicode.ambiguous_as_wide', True)
pd.set_option('display.unicode.east_asian_width', True)

pe = pd.read_csv('./pe.csv', header=0, sep=',', usecols=[0, 1, 4])
pb = pd.read_csv('./pb.csv', header=0, sep=',', usecols=[0, 1, 4])

pe.insert(loc = 3, column = 'PE温度', value=0)
pe.insert(loc = 4, column = 'PB(加权平均值)', value=0)
pe.insert(loc = 5, column = 'PB温度', value=0)

pe['PB(加权平均值)']=pb['PB(加权平均值)']

pe['PE温度'] = pe['PE-TTM(加权平均值)'].rank(pct=True)
pe['PB温度'] = pb['PB(加权平均值)'].rank(pct=True)

pe.insert(loc = 6, column = '长投温度', value=0)

line=0
while line < pe.shape[0]:
    pe.loc[line, '长投温度'] = (pe.at[line, 'PE温度'] * 100 + pe.at[line, 'PB温度'] * 100) / 2
    line += 1

print(pe.head())
```



## 八月秋高风怒号

#### 定投

- 定期定额
  - 收益打折扣
  - 不论价格高低，每次定额投入，购买份额收到投资日的市值影响
- ⭐定期不定额
  - 根据估值调整投入的金额
  - ⭐容易在低估值的时候买到更多份额
- ⭐Tips：
  - 设定提醒日
  - 不要在定投过程中加入自己的主观判断
  - 投资收益投入再利用（手动红利再投）

#### 确定投资金额

- 通过投资年限和目标金额计算每月定投的金额

```py
excel:
	=pmt(预期年化收益率 / 12, 投资年限 * 12, 0, 目标金额, 1)

python3:
	import numpy
	numpy.pmt(预期年化收益率 / 12, 投资年限 * 12, 0, 目标金额, 1)
```

#### 没有目标的投资计划制定范例

- 1. 先盘点存量金额(自己的存款)
  2. 存量金额里排除掉作为应急资金的部分
     - 应急资金应能够覆盖**3-6个月**的基本日常支出
     - 可以放在货币基金里（余额宝）
  3. 剩下的资金平摊到20个月中定投（10-30个月，为了平摊成本，时间应该放长）
  4. 确定增量资金
     - 增量资金 = 每月工资 - 基本支出 - 保险支出（当年保费/12）
  5. 风险系数评估
     - (风险系数)% = 100 - (当年年龄)
     - 如果相对保守。可以把100换成80，如果可以承担更大的风险，可以把100换成110
  6. 每月定投金额 = （存量金额 + 增量资金）* 风险系数百分比

#### 选择指数基金

- 选择温度最低的指数

#### 不定额投资

> 投资金额百分比：[卖出]< (0 = 持有)< [买入]

```
		温度  00 < 10 < 20 < 25 < 30 < 40 < 50
投资金额百分比 100 < 80 < 60 < 50 < 0 < -50 < -100
```

#### **指数切换**

- 当定投时出现比目前持有的指数基金温度更低(3°)的指数时，需要及时将定投目标切换到温度更低的指数基金
- 而原本在定投的基金可以继续持有，直到满足卖出条件，然后将其卖出即可
- 如果各种指数都不适合投资，可以将当前的资金以及获利转投货币基金，等待下次投资机遇



## 九重城阙烟尘生

#### 场内外投资区别

- 场内需要开通股票账户，手续费较低

- 场外可以通过各种去到进行交易，不需要开户，手续费较高

- 场内应该注意费率的细节，同时能在场内外交易的基金优先选场内

- 场内外交易对象不同，场内是与其他投资者根据供求关系用变动的价格交易基金；场外则是跟基金公司申购或赎回，没有第三方参与，场外一天基金只有一个价格（基金单位净值），既当天收市后基金的价格

  - > 基金单位净值 = (基金资产总市值 - 当天的各类成本) / 发行的总份额数量

- 场内外交易的效率不一样，场内交易成功后，当天确认份额，次日可以卖出，卖出后资金当天回到账户；场外如果在下午3点后申购，则要交易次日才能确认买入份额，而赎回后还要等2-4个工作日才能回到账户上

- 场内基金大多是现金分红，现金直接转入投资者账户；场外可以选择现金分红和红利再投，红利再投是指将分红直换算成更多的基金份额，红利再投购买的份额不需要再缴纳手续费

- 场内只能自己手动定投，而场外交易有多种方式设置自动定投。



## 十年辛苦不寻常

#### 心理建设

- 钱多钱少都可以投，相信定投，坚持就是胜利
- 只用"闲钱"投资
  - 绝对**不能**将所有钱都用来投资
  - 留足至少三个月的备用金，如果有配置保险，也需要排除保险支出，剩下的钱才用于投资
- 投机心理不可有
  - 没人可以预言股市的走势
  - 不要迷信“短线操作”
  - 投资都是**长期坚持**才能修成正果
- 喜涨厌跌不可有
  - 理性对待股市波动，坚持正确的投资策略
  - 估价低时坚持定投，上涨时赚取合理的收益
- 学会止盈，绝对不要被能力外的收益冲昏头脑，不懂的 不要碰！

#### 资产配置

- 债券和股权合理配置（初级入门级 如1：1）
  - 债权风险较低，股权风险更高
  - 资产配置可以减少亏损，但也会降低收益，而减少亏损往往是更重要的
  - 每年做一次平衡，让其比例回归初始值（1：1）
