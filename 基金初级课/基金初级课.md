# 基金

## 一蓑烟雨任平生

### 从名字了解基金

- 基金的名字大多由3部分组成： 公司名称 + 投资方向 or 基金特点 + 基金类型

  - 通俗点说，就是这只基金从哪里来，投资了什么，还有他的的类型是什么。

  - ```js
    // 天弘安康养老混合 代码 420009
    	天弘基金管理有限公司 养老方向
    // 国泰沪深300 代码 020011
        国泰基金管理有限公司 跟踪"沪深 300"指数
    // 嘉实沪港深精选股票 代码 001878
        嘉实基金管理有限公司 沪港深
    ```

- 基金名字后面的ABC

  - 货币基金
    - 如果基金的名字里面带A的后缀，一般申购门栏比较低，面向的是普通的投资者。
    - 后缀为B的，一般申购门槛都比较高，主要面向的是资金量大的用户或者机构投资者。
    - 后缀为C的，可以理解为和B的一样。申购门槛比较高，B和C区别不明显。
    - 如果出现了D类，要么即使申购门槛更高了，要么是某个公司指定了一个交易平台，专门办理申购赎回，然后基金名字后面加了个D。
    - 如果出现E。表示这个基金只在指定的机构进行代销，比如互联网平台；还有一种情况是它是交易型货币基金，是场内交易的。
  - 债券基金的`abc`主要在于收费的方式不同。
    - 他们都是根据申购费，收取的时间不同来划分。
    - 债券基金中A类，一般是`前端收取申购费`。就是买的时候就收费。
      - 基金公司有时为了吸引土豪加入，会对申购金额大的提供一定幅度的优惠
    - B类呢。一般是`后端收取申购费`，就是买的时候先不扣费。等赎回的时候才一次性收取。费率则随你持有基金的时间长短来决定
    - C类呢。一般会免去申购费。但是很有可能这只基存在"销售服务费率"。也就是用一个费率代替另一个费率而已。
    - 总结：如果是短期投资（1-2年），应该选择C类债券。如果是长期持有（5年或5年以上）可以选择B类。如果是不知道投资多久。那就选择A类把。实际中，还是要根据实际情况比较再做选择噢

    2. 红利再投资。



## 三山半落青山外

#### 筛选货币基金的四大指标

- `收益率`
  - 一般情况下，货币基金的收益率可以达到2%-4%。
  - 货币基金可以通过"每万份收益"和"7日年化"两个指标判断
    - 万份收益：就是指按上一日或上一个交易日的收益率，一万元本金能够赚多少钱。
      - 投资1千元。当天盈利0.2元。（`万份收益 = 0.2 / 1000 * 10000 = 2`）
    - 七日年化收益则是指，根据过去7天的收益总和，计算出的年花收益率。（）
      - 投资1000，过去七天赚了1元。（`七日年化收益 = 1 / 7 * 365 / 1000 * 100% ≈ 5.21%`）
    - 一般来说，货币基金的万份收益再1左右，7日年化在3%左右。
- `流动性`
  - 货币基金的流动性是指如果我们卖出这支基金后，自己多久能回到我们的账上。
  - 一般来说，有T+0 和 T+1两种交割方式。
- `成立时间`
  - 货币基金的成立时间，3-5年以上比较好
  - 如果一只基金太年的话，就没有历史的收益率等数据作为参考，从而对它做出合理的评估。相比较一些老牌基金，年轻的基金没有经过市场的磨练，隐藏的风险蛮大的。
- `基金规模`
  - 基金规模适中最好，一般不小于20亿，最大不要超过2000亿。货币基金的规模越大，就意味着这只基金中的钱越多
  - 规模越大的货币基金，有充足的资金应对赎回，也能留有更多的资金来投资
  - 规模越大的货币基金，因为资金问题无法正常运作而被清盘的风险也会比较低
  - `但是`规模太大的货币基金，可选择的投资品受限，收益反而会下滑。

- 由专业的基金经理和团队来帮你投资。

- 个人投资者很难做到分散投资。降低风险

- `成立时间`
  - 3-5年以上。
- `收益率`

- `规模`
  - 规模适中，一般在5-100亿。
- `费率`
  - 越低越好。
- `基金经理的更换频率`
  - 更换频率越低越好。

## 四面帘垂碧玉钩

#### 筛选混合型基金的七大指标

- `收益率`
  - 累计3年以上的收益率，优先选择收益率更高的
- `成立时间`
  - 成立时间在3年以上的，更加稳健
- `基金规模`
  - 混合基金规模适中即可。建议2亿-100亿之间。
- `手续费`
  - 越低越好
- `基金经理的更换频率`
  - 越低越好
- `基金公司的盈利能力`
  - 一是看它在同类基金公司中的排名。只需要选择排名靠前的即可
  - 二是看它盈利是否跑赢沪深300指数，因为沪深300可以大致代表大盘的走势
- `基金经理的选股择时能力`（排除指标）
  - 基金经理的选股择时能力，主要参考四个指标：从业情况、评分、星级和管理基金的收益排名
  - 从业情况主要看这个基金任职多久，如果小于3年不建议挑选。评分高于50分比较有保障，星级越高越好，四级以上最好。管理的基金收益排名没有跑赢同类平均水平的。也可以排除。

#### 选择纯债基金（纯债我自己也不是很了解）

- 百度"晨星网"

- 点击基金工具下面的"基金筛选器"

- 点选"纯债基金"，点选"3 星以上（包括 3 星）"三年评级和五年评级都选，然后点击“查询”按钮

- `纯债基金的买卖时机和“市场利率”密切相关。`

  - 市场利率： 是市场资金借贷成本的真实反映，能够及时反映短期市场利率的指标有银行间同业拆借利率、国债回购利率等。

  - 简单说：就是纯债基金的买卖时机和市场利率有关就好。

  - 其实，市场利率完全可以参考市场无风险收益率。也就是`在当前市场上，你把钱借给别人，在没有风险的情况下可以获得收益率。`

  - 市场利率一般参考10年期国债利率。（一下市场利率代表10年期国债利率）

    - 这是因为中长期债券可以更加细致的划分为中期债券和长期债券，10 年是中期债券和长期债券的分界点，期限短于10 年的债券收益较低，而超过 10 年的债券买的人又很 少，10 年期刚刚好，市面上几乎所有的投资品类都会跟它比较，所以 10 年期国债利率是一个非常有代表性的利率。

    - `市场利率上升，债券价格下跌;`

    - `市场利率下降，债券价格上升；`

    - ```js
      // 举个例子
      比如今年 2 月市场利率是 3%，二狗公司发行了债券，利息4%；到了 6 月份，市场利率升高到 3.5%，没有风险就可以获得 3.5%的收益，二狗公司的债券 4%的收益看起来没啥吸引力了呢。有些人就会觉得安稳获得 3.5%更划算，那
      么，那些买了二狗公司债券的人就会卖出债券，直到它的收益率重新变得有吸引力为止。从这个小例子就可以看到，随着市场利率从 3%升到3.5%，债券的价格是下跌的。同样的道理，市场利率下跌时，债券的价格会上涨
      ```

    - 那么当我们投资纯债基金的时候，肯定希望买入之后债券价格升高，也就是希望接下来市场利率下降。

    - ```js
      根据历史统计，中国 10 年期国债收益率的平均值在 3.3%-3.5%，这就是市场利率的平均值，由于市场利率波动小，比较稳定，这个平均值也是长期稳定的。
      所以我们只要拿现在的市场利率（也就是 10 年期国债收益率）和 3.5%比较就可以判断是否应该买入纯债基金：
      ```

      债基金；`

    - `当前市场利率（等于 10 年期国债收益率）<3%，卖出纯债
      基金，买入货币基金`

## 五陵少年争缠头

#### 指数基金

- 被动型基金
  - 基金经理不主动寻求超越市场表现，选取特定的股票作为投资对象，试图复制整个市场的收益率。长期来看，相比较主动型基金，被动型基金的收益更高更稳定
- 指数
  - 指数，其实就是一个选股规则，它按照某个规则挑选出一篮子股票，由专业机构通过复杂的计算，得出一个代表这篮子股票的平均价格，然后用来反映市场上，这一类股票的价格水平
  - 我们把这个平均价格称为"指数点位"，把放在篮子里的这些股票，称为这只指数的成分股。
  - 指数的成分股事可以变化的，一单某只股票不能满足原先的规则，就会被替换掉
- A股市场主要指数
  - 上证50指数
  - 沪深300指数
  - 中证500指数
  - 创业板指数
  - 红利指数

#### 指数基金的优点

- 灭绝人性
  - 只需要直接跟踪和复制指数
- 永垂不朽
  - 组成指数的公司不符合标准了，就会有新的公司代替
- 笑到最后
  - 只要经济是持续向好的，指数基金就有投资的价值

#### 指数基金的误差

- 指数基金本质是股票型基金，它依然存在风险，虽然是跟踪和复制指数来调整篮子里的股票组合。但是基金经理如果复制得有偏差，或者没有及时跟着指数调整，也会带来收益方面的误差风险，指数基金规模小或者运作历史不长，也会有风险
- 在天天基金网上，指数基金详情中可以看到`跟踪误差`，越小越好
