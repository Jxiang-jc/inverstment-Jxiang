# 基金

## 一眼看透基金

### 从名字了解基金

- 基金的名字大多由3部分组成： 公司名称 + 投资方向 or 基金特点 + 基金类型

  - 通俗点说，就是这只基金从哪里来，投资了什么，还有他的的类型是什么。

  - ```js
    // 天弘安康养老混合 代码 420009
    	天弘基金管理有限公司 养老方向
    // 国泰沪深300 代码 020011
        国泰基金管理有限公司 跟踪"沪深 300"指数
    // 嘉实沪港深精选股票 代码 001878
        嘉实基金管理有限公司 沪港深
    ```

- 基金名字后面的ABC

  - 货币基金
    - 如果基金的名字里面带A的后缀，一般申购门栏比较低，面向的是普通的投资者。
    - 后缀为B的，一般申购门槛都比较高，主要面向的是资金量大的用户或者机构投资者。
    - 后缀为C的，可以理解为和B的一样。申购门槛比较高，B和C区别不明显。
    - 如果出现了D类，要么即使申购门槛更高了，要么是某个公司指定了一个交易平台，专门办理申购赎回，然后基金名字后面加了个D。
    - 如果出现E。表示这个基金只在指定的机构进行代销，比如互联网平台；还有一种情况是它是交易型货币基金，是场内交易的。
  - 债券基金的`abc`主要在于收费的方式不同。
    - 他们都是根据申购费，收取的时间不同来划分。
    - 债券基金中A类，一般是`前端收取申购费`。就是买的时候就收费。
      - 基金公司有时为了吸引土豪加入，会对申购金额大的提供一定幅度的优惠
    - B类呢。一般是`后端收取申购费`，就是买的时候先不扣费。等赎回的时候才一次性收取。费率则随你持有基金的时间长短来决定
    - C类呢。一般会免去申购费。但是很有可能这只基存在"销售服务费率"。也就是用一个费率代替另一个费率而已。
    - 总结：如果是短期投资（1-2年），应该选择C类债券。如果是长期持有（5年或5年以上）可以选择B类。如果是不知道投资多久。那就选择A类把。实际中，还是要根据实际情况比较再做选择噢

#### 基金的收益与分配

- 基金的收益构成
  - 1. 买卖证券的差价。（投资品低买高卖赚的收益）
    2. 基金投资所得债券利息、红利、股息。
    3. 银行存款利息。
    4. 其他合法收入。
- 基金收益的分配方式
  - 1. 现金分红
    2. 红利再投资。



## 三山半落青山外

#### 筛选货币基金的四大指标

- 收益率
  - 一般情况下，货币基金的收益率可以达到2%-4%。
  - 货币基金可以通过"每万份收益"和"7日年化"两个指标判断
    - 万份收益：就是指按上一日或上一个交易日的收益率，一万元本金能够赚多少钱。
      - 投资1千元。当天盈利0.2元。（`万份收益 = 0.2 / 1000 * 10000 = 2`）
    - 七日年化收益则是指，根据过去7天的收益总和，计算出的年花收益率。（）
      - 投资1000，过去七天赚了1元。（`七日年化收益 = 1 / 7 * 365 / 1000 * 100% ≈ 5.21%`）
    - 一般来说，货币基金的万份收益再1左右，7日年化在3%左右。
- 流动性
  - 货币基金的流动性是指如果我们卖出这支基金后，自己多久能回到我们的账上。
  - 一般来说，有T+0 和 T+1两种交割方式。
- 成立时间
  - 货币基金的成立时间，3-5年以上比较好
  - 如果一只基金太年的话，就没有历史的收益率等数据作为参考，从而对它做出合理的评估。相比较一些老牌基金，年轻的基金没有经过市场的磨练，隐藏的风险蛮大的。
- 基金规模
  - 基金规模适中最好，一般不小于20亿，最大不要超过2000亿。货币基金的规模越大，就意味着这只基金中的钱越多
  - 规模越大的货币基金，有充足的资金应对赎回，也能留有更多的资金来投资
  - 规模越大的货币基金，因为资金问题无法正常运作而被清盘的风险也会比较低
  - `但是`规模太大的货币基金，可选择的投资品受限，收益反而会下滑。

#### 货币基金的筛选方法

- 1. 天天基金网
  2. 点击 "基金排行"
  3. 选择"货币基金排行"， 并且选择"100元起（A类）"。把筛选出来的基金按照近3年的收益率从高到低排序，然后选择前10家
  4. 然后逐一分析每只基金的成立时间、基金规模、流动性、收益率这四项指标

#### 债券基金的独有优势

- 由专业的基金经理和团队来帮你投资。
- 个人投资者资金有限。直接参与债券市场。会受到一些限制
- 个人投资者很难做到分散投资。降低风险

#### 筛选债券基金的五大指标

- 成立时间
  - 3-5年以上。
- 收益率
  - 看累计3年以上的收益率，优先选择收益率更高的。
- 规模
  - 规模适中，一般在5-100亿。
- 费率
  - 越低越好。
- 基金经理的更换频率
  - 更换频率越低越好。

#### 债券基金的筛选方法

- 1. 天天基金网
  2. 点击 "基金筛选"
  3. 选择 "债券型"
  4. 选择成立年限>7年的。其次。看累计3年以上的收益率，优先选择收益率更高的。
